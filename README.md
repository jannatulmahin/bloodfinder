# Bloodfinder 🩸

A Django-based Blood Finder web application that helps users find blood donors, register as donors, and request blood easily.

---

## 📌 Features

- User can register as a blood donor
- View available donors
- Request blood from donors
- Simple and clean user interface
- Built using Django framework

---

## 🛠️ Technologies Used

- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite3

---

bloodfinder/

## ⚙️ Installation & Setup

1. Clone the repository:
git clone https://github.com/jannatulmahin/bloodfinder.git
cd bloodfinder

2. Create virtual environment:
python -m venv venv

3. Activate virtual environment:
venv\Scripts\activate

4. Install dependencies:
pip install django

5. Apply migrations:
python manage.py migrate

6. Run the server:
python manage.py runserver

7. Open in browser:
http://127.0.0.1:8000/

# Bloodfinder

A Django-based Blood Finder web application where users can register as donors, search for donors, and request blood.

## Features

- Donor registration
- Donor list view
- Blood request form
- Blood request list
- Django-based project structure
- Static files and templates support

## Technologies Used

- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite3
